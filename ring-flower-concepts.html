Niche Tag Display Concepts

Three approaches for fanning niches out from the creator ring.
Using Andy Neal's tags as sample data. Click "Replay" to re-trigger the animation.

A. Spiral Bloom

Tags spiral outward from the avatar using a golden-angle distribution (≈137.5°). Each tag is placed at increasing radius, creating a sunflower-like pattern. Category-colored tags cluster naturally. Thin dashed stems connect back to center.

B. Branching Bouquet

Each category forms a "branch" radiating outward from the avatar. Tags cascade along each branch, creating a bouquet silhouette. Categories are distributed in a semicircle on the left, with thin connecting stems. Category labels act as branch headers.

C. Petal Arc

Tags flow in a sweeping arc from upper-left to lower-left of the avatar. Each category occupies a segment of the arc, with tags stacked at slight offsets within each segment. The effect is like a crescent of petals wrapping around the creator. Each category has a small label along the inner edge.
